# TherapEase

TherapEase is a secure, therapist-focused web application designed to reduce the administrative burden of running a small therapy practice. It brings core practice management tools into one platform, replacing the need for multiple disconnected applications.

The system allows therapists to manage clients, appointments, notes and payments from a single dashboard, improving efficiency and creating a more streamlined workflow.

## Technologies Used
- **Frontend:** React.js  
- **Backend:** Node.js with Express  
- **Database:** MongoDB  
- **Payments:** Stripe
- **Video Calling:** Zoom
- **Automatic Emails:** Nodemailer  
- **Automatic SMS:** Twilio  

Further setup and installation instructions will be added as development progresses.
